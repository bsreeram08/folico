# Code of Conduct

Folico uses a simple rule: be clear, kind, and useful.

## Expected Behavior

- Assume good intent.
- Give actionable feedback.
- Keep discussions focused on the product, code, and user safety.
- Respect that people use different macOS versions, workflows, and accessibility settings.

## Unacceptable Behavior

- Harassment or personal attacks
- Dismissive or hostile review comments
- Publishing private user data or paths
- Pressuring maintainers for unpaid support

## Enforcement

Maintainers may edit, hide, or close issues and pull requests that violate this code.
